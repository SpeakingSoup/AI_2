# Лабораторная работа №2: "Обучающая, проверочная и тестовая выборки. Переобучение нейронных сетей"

Данный репозиторий содержит файлы выполненной лабораторной работы №2

## Содержание

- *lite* – задание уровня Lite. Цель – создать таблицу точности из 9 экспериментов, основанных на моделях, распознающих виды одежды и аксессуаров;
- *lite_pytorch* – задание уровня Lite, выполненное с помощью библиотеки pytorch;
- *pro* – задание уровня Pro. Цель – выполнить загрузку, подготовку и предобработку данных модели классификации вин; необходимо добиться точности выще 94%;
- *ultra_pro* – задание уровна Ultra Pro. Цель – подготовить данные базы пассажиров автобуса и добиться точности выше 85%.
